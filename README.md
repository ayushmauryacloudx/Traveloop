# Traveloop Full-Stack Project

This project has been restructured into a full-stack MERN application (MongoDB, Express.js, Vanilla HTML/JS, Node.js).

## Project Structure

- `frontend/`: Contains all HTML, CSS, and vanilla JS logic. 
- `backend/`: Contains the Express server, MongoDB schemas, and API logic.

## How to Run

1. Open a terminal in the `backend/` folder.
2. Run `npm install` to install dependencies.
3. Make sure MongoDB is running locally on port 27017 or update `MONGO_URI` in `backend/.env`.
4. Run `npm run dev` to start the backend server on port 5000.
5. Open any of the HTML files in the `frontend/` folder in your browser (preferably via a Live Server extension).
